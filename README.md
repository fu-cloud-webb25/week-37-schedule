# Schema, vecka 37
###### Utveckling och driftsättning i molnmiljö, vecka 3 av 6

## Introduktion

Ett API blir sällan användbart utan någonstans att lagra data. Den här veckan kopplar vi ihop våra serverlessfunktioner med DynamoDB och bygger våra första datadrivna molnapplikationer.

Vi lär oss hur NoSQL-databaser fungerar, hur data modelleras i DynamoDB och hur våra Lambda-funktioner kan läsa, skapa, uppdatera och ta bort information. Fokus ligger på att förstå hur databasen påverkar både arkitektur, prestanda och skalbarhet.

## Mål för veckan:

1. Förstå hur DynamoDB fungerar.
2. Förstå skillnaden mellan SQL och NoSQL.
3. Modellera data för DynamoDB.
4. Integrera Lambda med DynamoDB.
5. Bygga CRUD-funktionalitet.
6. Förstå hur olika datamodeller påverkar skalbarheten.
